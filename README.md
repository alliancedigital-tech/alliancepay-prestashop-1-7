# Платіжний шлюз AlliancePay для Prestashop 1.7.x

![AlliancePay Logo](https://alb.ua/_next/static/media/Alliance_logo.25cb4c66.svg)

## Огляд
Офіційний модуль платіжного шлюзу **AlliancePay** для **Prestashop**. Цей модуль дозволяє безперешкодно інтегрувати AlliancePay у ваш інтернет-магазин, забезпечуючи безпечний, швидкий та надійний процес оплати для ваших клієнтів.

## Основні можливості
* Безпечна обробка платежів через API AlliancePay.
* Підтримка різних валют та методів оплати Visa/Mastercard/GooglePay/ApplePay.
* Автоматична синхронізація статусів платежів (webhook / callback).
* Просте налаштування та зручний інтерфейс в адмін-панелі.

## Вимоги
* PHP 7.3 або вище
* PHP розширення: gmp, hash, openssl, sodium
* Prestashop версії 1.7.6.x+
* Активний обліковий запис мерчанта AlliancePay.
